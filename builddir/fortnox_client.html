Fortnox client

FortnoxClient.cls is the APEX class used to make external calls to the Fortnox API. Fortnox provides both API DOCS and General Guidelines.

Endpoints

The base URL for the Fortnox API: https://api.fortnox.se/3.

Below headers are used for authentication and to get response in the desired format:

Key

Value

Accept

application/json

Content-Type

application/json

Client-Secret

clientSecret()

Access-Token

accessToken()
